# What happens when you type google.com in your browser and press Enter

This project explains the full web infrastructure flow that happens when a user types `https://www.google.com` in a browser and presses Enter.

The blog post covers:

- DNS request
- TCP/IP
- Firewall
- HTTPS/SSL
- Load balancer
- Web server
- Application server
- Database
